# HelpinOut [React Native]

## How to start development
- clone the repo

execute the commands as follows
  - npm install
  - cd /ios
  - pod install
  - cd root directory
  - npx react-native run-ios (if react-native cli is not installed)
- react-native run-ios (if react-native cli is  installed)
